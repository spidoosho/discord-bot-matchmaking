Welcome to ValoJS REST API

This is an extension of third-party matchmaking Discord Bot ValoJS. You can find more about the Bot here.

Usage

Before using the API, you need to obtain Discord server ID and Discord user ID. Check this tutorial on how to find them.

Leaderboard

Leaderboard returns leadeboard ID and player data in a descending order based on player's ratings. If you want to get JSON or as a text, at the end of the link append /json or /text respectively.

https://valojs.onrender.com/${DISCORD_SERVER_ID}

Player data

Player data returns leadeboard ID and player data. If you want to get JSON or as a text, at the end of the link append /json or /text respectively.

https://valojs.onrender.com/${DISCORD_SERVER_ID}/${DISCORD_USER_ID}
